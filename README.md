mini-bookstore-web

A mini bookstore web application built with HTML, CSS, and JavaScript.
This project allows users to register, log in, view all books, and search for books by ISBN, author, or title. It simulates a simple front-end bookstore experience with asynchronous operations using Promises and callbacks.

Features

User Registration – Create a new account with a username and password.

User Login – Login using registered credentials.

View All Books – Display all available books in the shop.

Search by ISBN – Find a book by its ISBN using Promises.

Search by Author – Find all books by a specific author using Promises.

Search by Title – Find books matching a title using Promises.

Fully styled HTML/CSS front-end.

Technologies Used

HTML5 – Structure of the web pages.

CSS3 – Styling the UI for a modern look.

JavaScript – Implementing functionality, async callbacks, and Promises.
